Purview User Behavior Investigator

PurviewUserBehaviorInvestigator
Microsoft Purview の DLP/IRM アラート発砲後に、対象ユーザーの行動履歴を調査するカスタムエージェントです。 ユーザー UPN とオプションのアラート情報を受け取り、アラート前後1週間の Exchange/Teams・メール・クラウドアプリ利用状況を Sentinel および Defender から収集・分析し、社外通信パターンと不審なクラウドストレージ利用を含む事実ベースの Markdown 行動調査レポートを生成します。

Investigate User Behavior After DLP/IRM Alert
InvestigateUserBehavior

AlertInfo DLP/IRM アラートの詳細情報（ポリシー名、重大度、トリガーとなった操作など） No
